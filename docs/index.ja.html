ClipboardHistory for macOS

メニューバーから使える、ローカル完結の clipboard history。

ClipboardHistory は、最近コピーしたテキストや画像を、コンパクトなフローティングパネルから すばやく呼び戻せる macOS アプリです。pin、編集、再コピー、paste-back に対応します。

Universal

通常はこちらを推奨します。Apple Silicon / Intel の両方で動きます。

Universal をダウンロード
Apple Silicon

M1、M2、M3 以降の Apple Silicon 向け thin build です。

Apple Silicon をダウンロード
Intel

Intel Mac 向け thin build です。Intel 専用が必要な場合のみ使ってください。

Intel をダウンロード

インストール

  • 使っている Mac に合う build をダウンロードします。
  • ClipboardHistory.app を展開して /Applications に移動します。
  • 一度起動し、必要なら Accessibility 権限を許可します。
  • ショートカットや launch at login は Settings から変更できます。
  • /Applications に入れてから Launch at Login を有効にしてください。

対応範囲

  • macOS 14 以降
  • テキストと画像の clipboard history
  • Apple Silicon / Intel 向け release artifact
  • ローカル完結。アカウントや sync はありません。

Project

source、issue、release note は GitHub にあります。checksum は各 release の SHA256SUMS.txt に含まれます。

特に理由がなければ Universal build を使ってください。thin build は architecture 固定が必要な場合向けです。

panel hotkey と paste-back のために Accessibility permission が必要になることがあります。
