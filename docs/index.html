ClipboardHistory for macOS

Fast local clipboard history, built for the menu bar.

ClipboardHistory keeps recent text and image snippets available from a compact floating panel, with pinning, editing, paste-back, and local-only storage.

Universal

Recommended for most people. Works on Apple Silicon and Intel Macs.

Install

  • Download the build that matches your Mac.
  • Unzip and move ClipboardHistory.app into /Applications.
  • Launch once, then grant Accessibility permission if prompted.
  • Use the settings window to configure shortcuts and launch at login.

Compatibility

  • macOS 14 or later
  • Text and image clipboard history
  • Apple Silicon and Intel release artifacts
  • Local-only workflow, no sync or account setup

Project

Source, issues, and release notes live on GitHub. The latest checksums are published in each release as SHA256SUMS.txt.

Tip: the Universal build is the default recommendation. Use the thin builds only when you specifically want a smaller architecture-specific bundle.
